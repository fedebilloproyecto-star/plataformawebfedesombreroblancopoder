Bloc de notas - Características

  • Editor de texto plano con soporte para múltiples formatos
  • Capacidad de guardar y cargar documentos
  • Búsqueda y reemplazo de texto
  • Soporte para diferentes codificaciones de texto
  • Integración con el portapapeles del sistema
  • Soporte para múltiples pestañas
  • Resaltado de sintaxis para programación

Calculadora - Características

  • Operaciones aritméticas básicas (+, -, ×, ÷)
  • Cálculo de porcentajes
  • Funciones científicas (raíz cuadrada, potencias)
  • Memoria para almacenar resultados
  • Historial de operaciones recientes
  • Modo programador con conversión entre sistemas numéricos
  • Gráficos de funciones matemáticas

Navegador Web - Características

  • Motor de renderizado WebKit/Blink
  • Soporte para HTML5, CSS3 y JavaScript
  • Pestañas múltiples para navegación simultánea
  • Marcadores y historial de navegación
  • Modo incógnito para navegación privada
  • Sincronización con la nube
  • Extensión de navegador para funcionalidades adicionales

Reproductor de Música - Características

  • Soporte para múltiples formatos (MP3, WAV, FLAC, AAC)
  • Ecualizador con presets personalizables
  • Creación y gestión de listas de reproducción
  • Visualizador de espectro de audio
  • Integración con biblioteca multimedia
  • Soporte para streaming online
  • Letras de canciones sincronizadas

Cámara - Características

  • Resolución de hasta 12MP con ajustes manuales
  • Modos de disparo (automático, deportes, nocturno, retrato)
  • Balance de blancos y ajuste de exposición
  • Enfoque automático y manual
  • Galería integrada con organización por fechas
  • Edición básica de fotos (recortar, rotar, ajustes)
  • Soporte para RAW y HDR

Microsoft Outlook - Características

  • Gestión de correo electrónico profesional
  • Calendario integrado para gestión de tiempo
  • Contactos y agenda
  • Bandeja de entrada inteligente con filtros
  • Integración con Microsoft 365
  • Búsqueda avanzada con filtros
  • Soporte para múltiples cuentas

Microsoft OneDrive - Características

  • Almacenamiento en la nube seguro
  • Sincronización de archivos entre dispositivos
  • Compartición de archivos y carpetas
  • Historial de versiones de documentos
  • Integración con Office Online
  • Cifrado de datos en tránsito y en reposo
  • Soporte para trabajo colaborativo

Twitter - Características

  • Publicación de tweets (280 caracteres)
  • Timeline personalizado
  • Mensajes directos
  • Listas de usuarios
  • Soporte para imágenes, videos y GIFs
  • Notificaciones en tiempo real
  • Análisis de engagement

Google Drive - Características

  • 15GB de almacenamiento gratuito
  • Sincronización automática entre dispositivos
  • Acceso desde cualquier navegador
  • Integración con Google Docs, Sheets y Slides
  • Compartición avanzada de archivos
  • Búsqueda potente con filtros
  • Soporte para trabajo colaborativo en tiempo real

Notebook Avanzada

CPU (Procesador)

Intel Core i9-12900H de 14 núcleos (6 Performance + 8 Efficient) hasta 5.0GHz con 24MB de cache

GPU (Tarjeta Gráfica)

NVIDIA RTX 3080 Ti con 16GB GDDR6 y soporte para DLSS y Ray Tracing

RAM

32GB DDR5 a 4800MHz en configuración dual-channel (ampliable hasta 64GB)

Almacenamiento

1TB SSD NVMe PCIe 4.0 con velocidades de 7000/5000 MB/s (lectura/escritura) + ranura adicional para expansión

Placa Madre

Chipset Z690 con soporte para PCIe 5.0, Thunderbolt 4 y WiFi 6E

Pantalla

16" QHD+ (2560×1600) IPS 240Hz con 100% DCI-P3 y HDR1000

Refrigeración

Sistema de vapor chamber dual fan con 4 salidas de aire y 5 heat pipes

Smartphone Flagship

Snapdragon 8 Gen 2 de 4nm con CPU Octa-core (1x3.2GHz + 4x2.8GHz + 3x2.0GHz) y GPU Adreno 740

Memoria RAM

12GB LPDDR5X + RAM virtual de hasta 8GB adicionales

Almacenamiento

256GB UFS 4.0 con velocidades de 4200/2800 MB/s (lectura/escritura)

Pantalla

6.7" Dynamic AMOLED 2X QHD+ (3200×1440) con LTPO 1-120Hz, 1500 nits y protección Gorilla Glass Victus 2

Batería

5000mAh con carga rápida de 65W, carga inalámbrica de 15W y carga reversible de 5W

Cámaras

Triple sistema: Principal 200MP OIS, Ultra Wide 50MP y Telefoto 10MP con zoom óptico 3x

Conectividad

5G Sub-6GHz/mmWave, WiFi 6E, Bluetooth 5.3, NFC, UWB y GPS dual-band
